conclution

The objective of this project is to create and implement a project  for online bookshop.  This project  will allow users to create and maintain individual secured accounts, search the  Bookstore database for books, and make secured online credit card purchases. Users will also be able to contact site administrators. The project makes purchasing books quicker, easier, and more convenient.And the ERD and Database schema will help to understand the overall model of this procedure.
